This is the front page for the University of Michigan Library's GitHub organization

Welcome!

Members of the U-M Library's GitHub organization can access the private wiki. Others can browse the list of our public repositories.

Joining the U-M Library GitHub organization

The U-M Library GitHub organization is managed by Architecture and Engineering. Library staff can request to be added to the organization. When doing so, please include your GitHub account name.
